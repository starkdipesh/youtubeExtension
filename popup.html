🎬 YouTube Channel Restrictor

Restrict YouTube to a specific channel

Use the @handle from your channel URL (e.g., https://www.youtube.com/@tech-gamerz7292)

This name will show in notifications and overlays

✨ Features

  • ✅ Use @handle or Channel ID to filter
  • ✅ Auto-plays next video from allowed channel
  • ✅ Scans 'Up Next' sidebar for matching videos
  • ✅ Auto-plays latest video if none in sidebar
  • ✅ Works with YouTube's dynamic page loads (SPA)
  • ✅ Clean, non-intrusive UI with status overlays

⚠️ Important Notes

  • Videos from other channels will be blocked
  • Recommendations are filtered to the allowed channel only
  • This extension works on YouTube.com only
  • Reload any open YouTube tabs after changing settings
